Репозиторий с упрощённым примером DWH (Data Warehouse) инфраструктуры для тестирования MCP/Git-интеграций, поиска по репозиторию и анализа структуры проекта.

Содержит:

- SQL-модели витрин и staging-слоя
- ETL/ELT пайплайны
- примеры DAG/flows
- конфиги источников данных
- несколько веток с изменениями схем и пайплайнов
- набор тестовых патчей
